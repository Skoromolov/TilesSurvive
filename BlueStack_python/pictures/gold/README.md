# Изображения для золотодобычи

## Необходимые скриншоты

Поместите следующие изображения в эту папку:

| Файл | Описание | Пример |
|------|----------|--------|
| `events.png` | Кнопка "События" (Events) в интерфейсе игры | Иконка календаря или события |
| `rudnik.png` | Кнопка/иконка рудника в списке событий | Изображение рудника в меню событий |
| `rudnik_opened.png` | Открытый рудник (проверка что мы на странице рудника) | Заголовок или фоновое изображение рудника |
| `find.png` | Кнопка поиска рудника | Лупа или кнопка "Найти" |
| `my_rudnik.png` | Мой рудник (если уже есть активный) | Изображение вашего рудника |
| `return.png` | Кнопка "Отозвать" (Return) | Кнопка отозвать/вернуть |
| `grind.png` | Кнопка "Добыча" (Grind) | Кнопка начала добычи |
| `work.png` | Кнопка "Работа" (Work) | Кнопка начать работу |
| `go.png` | Кнопка "Поехать" (Go) | Кнопка отправить/поехать |

## Как сделать скриншоты

1. Запустите игру в BlueStacks
2. Перейдите в соответствующий экран
3. Сделайте скриншот нужного элемента
4. Обрежьте до минимального размера (только сам элемент)
5. Сохраните в формате PNG

## Рекомендации

- **Размер**: 50x50 - 200x200 пикселей
- **Формат**: PNG (поддерживает прозрачность)
- **Качество**: чёткое, без размытия
- **Язык**: русский (как в игре)

## Проверка

После добавления изображений запустите скрипт и проверьте логи:
```bash
python main.py
```

Если изображения не найдены, проверьте:
1. Правильность имени файла
2. Соответствие элемента на скриншоте
3. Чёткость и контрастность изображения
