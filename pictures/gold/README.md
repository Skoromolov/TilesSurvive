# Изображения для золотодобычи

## Необходимые скриншоты

Поместите следующие изображения в папку `pictures/gold/`:

| Файл | Описание | Пример |
|------|----------|--------|
| `events.png` | Кнопка "События" (Events) в интерфейсе игры | Иконка календаря или события |
| `rudnik.png` | Иконка рудника в списке событий | Изображение рудника в меню событий |
| `rudnik_opened.png` | Открытая таба рудника | Заголовок или фоновое изображение рудника |
| `select_level.png` | Виджет текущего уровня / вход в список уровней | "3 / уровень 6" с иконкой шахты |
| `lvl_1.png` ... `lvl_6.png` | Карточки уровней в списке выбора | Карточка с номером уровня и кнопкой "Перейти" |
| `current_lvl_1.png` ... `current_lvl_6.png` | Индикаторы текущего открытого уровня | Плашка/заголовок "Подземный золотой рудник — N-й этаж" |
| `current_raid_lvl_icon.png` | Иконка активной добычи для открытия деталей | Стрелка / значок уровня на экране "Мой рудник" |
| `find.png` | Кнопка поиска рудника | Лупа или кнопка "Найти" |
| `free_place.png` | Свободное место в результате поиска | Плашка "Свободное место" |
| `my_rudnik.png` | Экран "Мой рудник" | Персонаж с киркой и надпись "Мой рудник" |
| `return.png` | Кнопка "Отозвать" (Return) | Кнопка отозвать/вернуть |
| `return_boys.png` | Подтверждение отзыва отряда | Кнопка подтверждения |
| `grind.png` | Кнопка "Добыча" (Grind) | Кнопка начала добычи |
| `work.png` | Кнопка "Работа" (Work) | Кнопка начать работу |
| `go.png` | Кнопка "Поехать" (Go) | Кнопка отправить/поехать |

## Дополнительные / резервные элементы

| Файл | Описание |
|------|----------|
| `hand.png` | Указатель / направляющая рука |
| `confirm.png` | Кнопка "Подтвердить" |
| `finish.png` | Кнопка "Завершить" |

## Как сделать скриншоты

1. Запустите игру в BlueStacks.
2. Перейдите в соответствующий экран (события → рудник → список уровней).
3. Сделайте скриншот нужного элемента.
4. Обрежьте до минимального размера (только сам элемент).
5. Сохраните в формате PNG.

## Рекомендации

- **Размер**: 50×50 — 200×200 пикселей для кнопок; карточки уровней могут быть крупнее.
- **Формат**: PNG (поддерживает прозрачность).
- **Качество**: чёткое, без размытия.
- **Язык**: русский (как в игре).
- **Уровни**: для `lvl_X.png` обрезайте всю карточку целиком, включая кнопку "Перейти" внизу — бот кликает в нижнюю часть найденной карточки.

## Проверка

После добавления изображений запустите скрипт и проверьте логи:

```bash
python main.py
```

Если изображения не найдены, проверьте:
1. Правильность имени файла.
2. Соответствие элемента на скриншоте.
3. Чёткость и контрастность изображения.
4. Что `GOLD_ENABLED = True` и `GOLD_LEVEL` задан от 1 до 6.
